Graficando Datos De La Sonda

Graficando Datos De La Sonda en Timpo Real y Salvado en SQLite

10/05/2023

El siguiente paso después de montar la sonda es recibir los datos que esta envía por radio, para esto necesitamos escuchar el puerto serie de la antena de tierra en este caso usaré la librería serial de Python para leer el puerto.

import serial

PS = serial.Serial('COMX', 9600)
while 1:
	print(PS.readline())

El output que se obtiene de este simple script es muy incómodo de leer, porque le llega la información de la antena que es la mínima posible para reducir la cantidad de información enviada al máximo:

Ex:

1216,-87.85,102384.57,18.31 

1215,-83.85,101384.57,17.31

Es complicado tener en cuenta variaciones de los datos teniendo en cuenta que llega una nueva cadena cada segundo. Por tanto hacía falta una interfaz cómoda para leer los datos. Usé las librería matplotlib para graficar los datos y threading para la gestión de hilo que permiten crear varios procesos paralelos.

Por último queda almacenar la información en una base de datos, para la gestion de esta usé SQLite, y con un par de líneas en el código conseguí guardar los datos que llegan por serial, esto es útil para un posterior análisis de los datos.
